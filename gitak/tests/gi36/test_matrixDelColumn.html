Matrix Delete Column test, make sure record deletion works with delete column. Should be tested over HTTPS.
Test Matrix delete column
include testSetup.html  
jsxopen ${GIPATH}/shell.html?jsxapppath=${SAMPLESPATH}/test-matrix  
waitForElementPresent gi=tbbList 5000
assertElementPresent gi=tbbList  
click gi=tbbList  
waitForElementPresent gi=matrix1,rowbyindex,14 5000
assertElementPresent gi=matrix1,rowbyindex,14  
// Sort on delete column first. IE7 is reverse of other browsers
clickJsxMatrixHeader gi=matrix1,column,0  
pause 500  
assertElementPresent gi=matrix1,cellbyindex,14.0 // last row is index 14 of 0-14
// Delete Angola
clickJsxCellMask JsxMatrixCellId=matrix1.AO.0  
assertElementNotPresent gi=matrix1,cellbyindex,14.0 // row 14 was deleted
// Delete Azerbaijanen
assertElementPresent gi=matrix1,cellbyindex,13.0  
clickJsxCellMask JsxMatrixCellIndex=matrix1.13.0 //one row deleted only 0-13 left
assertElementNotPresent gi=matrix1,cellbyindex,13.0  
// Delete Albenia
assertElementPresent gi=matrix1,cellbyindex,1.0  
clickJsxCellMask JsxMatrixCellId=matrix1.AL.0  
assertElementNotPresent gi=matrix1,cellbyjsxid,AL.0
